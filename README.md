# 🍫 shoco

[shoco](https://github.com/Ed-von-Schleck/shoco) short strings compressor interface for Crystal

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     shoco:
       github: mentalblood0/shoco
   ```

2. Run `shards install`
